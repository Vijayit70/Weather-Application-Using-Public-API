## Description
This project is a responsive weather application that fetches real-time weather data from a public API and displays it dynamically based on user input.

## Features
- Search weather by city name
- Displays temperature, weather condition, humidity, and wind speed
- Real-time API data fetching
- Responsive and user-friendly UI
- Error handling for invalid city names

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## How to Run
1. Clone or download the project
2. Open `index.html` in a browser
3. Enter a city name and click **Get Weather**

## API Used
- OpenWeatherMap API

## Output
The app dynamically displays live weather details for the entered city.
